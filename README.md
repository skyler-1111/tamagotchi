**Bit-Bit**

----------------------------------------------

A Pomodoro timer where the pet dies if the user stops focusing.

--------------------------

**Mechanics**

Bit-Bit is a digital pet that requires constant attention. While a focus timer is active, the user "mines" coins. These coins are used in the shop to keep the pet alive or buy upgrades.

+Social Bar: Drops constantly. If it hits zero, the pet self-destructs.

+Sleep Mode: Slows social decay by 10x for longer focus sessions.

+Shop: Spend coins on "Auto-Petters" (10-minute social lockdown), new skins, or shades.

+Feed and Clean: You've to feed and clean it.

+Explosion: If the social meter empty, the pet explodes and the system requires a reboot.

**Design**

The UI uses a Neo-Brutalist aesthetic.

**Setup**

Bit-Bit is a single-file web application.

+Clone the repository.

+Open index.html in any modern browser.

+Start a focus session to begin mining.
